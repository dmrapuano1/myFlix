Documentation

This application will allow users to create accounts and add movies to their favorites list. Below are a list of actions any user can make.

Request URL URL Parameters HTTP method Example body input Example Response (current response)
none / n/a GET n/a 'Welcome to my page!'
Get a list of all movies in the system /movies n/a GET n/a ...{title:'Schindler's List',genre:'Drama',year:'1993',director:'Steven Spielberg'},{title:'The Shining',genre:'Horror',year:'1980',director:'Stanley Kubrick'},
Gets information on a singular movie /movies/:title :title - movie title you want to get information about GET n/a {title:'Schindler's List',genre:'Drama',year:'1993',director:'Steven Spielberg'}
Pulls information about genres /genres n/a GET n/a 'Details about each genre here'
Pull information about a specific director, currently pulls 1st movie in list with that director /directors/:director :director - director you want to get information about GET n/a {title:'Schindler's List',genre:'Drama',year:'1993',director:'Steven Spielberg'}
Registers account for new user /accounts/register n/a POST account name, favorites list 'Use this URL to create an account'
Updates the specified account as needed /accounts/:id/update :id - account id PUT account name, favorites list 'Use this URL to update your account with id: 425'
Deletes the specified account /users/:id :id - account id DELETE n/a 'Account will be deleted as soon as I write the code for it'
Adds a movie to the users favorites list /users/:id/:movie :id - account id
:movie - movie the user wants to add to the list		 ADD Inception (movie title) Adds 'Inception' to account with ID: 425
Removes a movie to the users favorites list /users/:id/:movie :id - account id
:movie - movie the user wants to remove from the list		 DELETE Inception (movie title) 'Inception' will be deleted from list of user with id of 425
Allows the user to read about how to use the API /documentation.html n/a GET n/a Loads this exact page
